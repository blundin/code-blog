Frequently Asked Questions

Click on the questions below to view each answer.

  • What does each setting do?

    The settings in Quick Grades give you control over your experience and give you exactly the information you want, when you want it.

    Screen Lock: Flipping this switch on will prevent your screen from dimming and the phone from locking as long as you are using the app. This is helpful while you are grading by keeping the screen active for your use. This can reduce your battery life while in use.

    Decimal Places: This number sets the number of decimal places to show when calculating grade percentages.

    Round up at: The settings specifies the decimal where the letter grade is rounded up. For example, if this value is set to '0.5' a 89.5% would be an 'A', and an 89.4% would be a B.

    The following options control the visibility of each column:

    Missed Questions: The number of questions missed on the assignment being graded.

    Correct Questions: The number correct out of the number possible.

    Score: The calculated percentage for the given score.

    Letter Grade: The calculated letter grade for the given score.
